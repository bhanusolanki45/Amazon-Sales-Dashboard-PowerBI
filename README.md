# Amazon-Sales-Dashboard-PowerBI
Interactive Power BI dashboard for Amazon product sales analysis – tracking YTD/QTR sales, product categories, top-performing products, monthly/weekly trends, and key performance metrics.

# Amazon Sales Dashboard - Power BI

![Dashboard Preview](https://github.com/bhanusolanki45/Amazon-Sales-Dashboard-PowerBI/blob/main/Aamzon%20Sales%20Dashboard.pdf)

## Project Overview
Interactive Power BI dashboard analyzing Amazon e-commerce sales data. Provides insights into year-to-date (YTD) and quarterly sales performance, product category breakdown, top products by sales and reviews, and temporal trends (monthly and weekly).

**Key Metrics Highlighted:**
- YTD Sales: $2.18M
- Quarterly Sales: $811.1K
- YTD Products Sold: 27.75K
- Total Reviews (Year): 19.42M
- Top categories: Men Shoes ($9.4M+ YTD), Camera, Men Clothes, Car Accessories, Toys
- Sales trends across months and weeks

## Key Insights
- Men Shoes dominate YTD sales contribution (43.18%)
- Seasonal/monthly sales patterns with peaks visible
- Top 5 products by sales and customer reviews identified
- Quarter-wise performance tracking

## Tools & Technologies
- **Microsoft Power BI Desktop** – Data modeling, DAX calculations, visualizations
- Power Query for data transformation and cleaning

## How to Use
1. Download `Amazon Sales Dashboard.pbix`
2. Open in Power BI Desktop (free from Microsoft)
3. Use filters (Product Category, Quarter) to explore interactively
4. Data is embedded – no external connections required

## Data Source
Public/sample Amazon sales dataset (commonly used for analytics portfolios and tutorials).
